Common questions: print vs return

  • Print just shows the HUMAN user a string so you can see what's going on.
  • Print does NOT affect how the function behaves!
  • Return is how a function returns value. This affects behavior!
  • If you don't have a return statement, then a function's value is undefined.

Common questions: print vs return

Cryptography: What is it?

  • Send and receive secret messages.
  • Encryption rule + plain text message = encrypted message.
  • The encryption rule must only be known by the sender & receiver.
  • Problem: to share a secret, you must first share a secret.

Cryptography: Secret sharing

  • Alice & Bob want to share a secret (a key).
  • Eve is a hacker and wants to get their secret.
  • Let's explain this with paint!
  • "Paint" === codes.

Cryptography: Mixing paints

Cryptography: Useful Properties

  • Unmixing is necessary for Eve to succeed
  • But unmixing is hard!
  • Alice & Bob don't need to unmix.

Cryptography: Paints === Math

  • One-way function: easy to mix (calculate), hard to unmix (reverse the calcluation).
  • Commutative: mixing in different order doesn't affect result.
  • No-one has found a PROVABLY one-way function! Best: long time to reverse.

Cryptography: Mod exponentiation

  • We reviewed mod yesterday: It's the remainder.
  • Choose a base g and modulus N. (base color, everyone knows).
  • Have a secret value x. (private color, nobody knows).
  • Calculate g^x mod N.

Cryptography: Example

  • Your base is 2, mod is 11 (public information)
  • You choose your x = 6.
  • 2 ^ 6 mod 11 = 64 mod 11 = 9 mod 11 = 9
  • Try it yourself!

Cryptography: Cracking mod exponentiation

  • If your N (the number you mod by) = 2^1024 (a big number!),
