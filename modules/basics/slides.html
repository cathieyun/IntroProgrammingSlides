What is Javascript?

  • Javascript is a programming language that acts as the logic for all websites. (demo)
  • We'll be learning Javascript because it is extremely easy to see your code in action.
  • What you'll learn today will be applicable to other programming languages too!

How can I do all this stuff myself?

The Building Blocks: Values

  • A value is the simplest unit of programming.
  • A value can have different types.
  • Different types of values can do different things,
    and are formatted different ways
  • Play around to find out how you can manipulate and combine values!

The Building Blocks: Variables

  • What does the word variable mean?
  • Variables are ways to store values, and can change their value.
  • Create variables by using the "var" keyword.
  • Store variables by using "=".

The Building Blocks: Functions

  • Functions are just actions.
  • You can define your own functions, or use pre-defined functions.
  • The Javascript reference website shows you how to use pre-defined functions.

The Building Blocks: Functions

  • In javascript, functions are values, just like strings and numbers!
  • Define your own function with the "var" and "function" keywords.
  • Whatever goes in the parenthesis of "function()" are the inputs to the function.

Getting Fancy: For loops

  • Get lazy - make your program do repetitions for you!
  • For loops are the most basic: do this thing i times.
  • Use "for (i=0; i < ___; i++)" to define loop range.

Getting Fancy: For In loops

  • Loop through an array in a more user-friendly way.
  • Acts the same as a for loop, but fetches value for you.

Getting Fancy: Conditionals
